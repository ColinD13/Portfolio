Basic Joke Web Service

This is a basic web service designed in Java using Springboot to run the application and Postman to test the Api calls. Tis web service has get, post, put, and delete capabilities along with basic authorization. The endpoints for each call are /joke and they all return a type ResponseEntity of the object Joke so that the output returns in Json format.
